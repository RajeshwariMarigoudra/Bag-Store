# Bags Store Website

This is a simple bags store website created using HTML, CSS, and Bootstrap. It displays different types of bags using Bootstrap cards and includes a navigation bar and footer.

## Project Structure

- `index.html`: The main HTML file.
- `css/styles.css`: Custom CSS for styling.
- `images/`: Folder containing images used in the project.

## How to Run

1. Clone or download the project.
2. Open the `index.html` file in your browser.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 4.5
